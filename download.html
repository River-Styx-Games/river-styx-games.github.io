Download the Game

Currently, this is the only place you can officially download the game. In the future, however, we plan to release it on several different platforms, including itch.io, Steam and/or the Google Play Store. Until then, please download the game from here, and share the project with your friends if you enjoy it!

This is only the Deemo version of the game, there is no full release yet as we are still working on development. That said, it should still be bug-free and completely enjoyable! There are sample features that are not available in the prologue, but will be fully implemented in the full version, including the Gallery and the Store. If you find any bugs, please report them to us! Find out how on the Contact Us page.

Install Instructions

Installing and playing the game is easy! Just download the game from one of the two buttons above, right click the zip folder, extract it, and run the file called Muses of Mayhem.exe. On some computers, the file may just be called Muses of Mayhem, but either way, it should have an icon other than the standard Windows exe icon. Run that, and enjoy the game!
